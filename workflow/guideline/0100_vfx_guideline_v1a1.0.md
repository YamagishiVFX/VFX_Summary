# 半角文字
- VFXツールの多くは英語圏で開発されている背景があり、データの名前に全角文字などを使うとバグの原因となる事があるので、**制作に直接使うデータ** のフォルダ名やファイル名なども含め、**極力半角文字を使う**。
- （UTF8：ユニコード文字の普及により、かなり改善されてつつある印象はある）
